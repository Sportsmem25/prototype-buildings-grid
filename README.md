# 2D scene prototype
Данный проект представляет собой 2D-систему размещения и удаления зданий на грид-сетке. 
Пользователь может выбирать тип здания через интерфейс, размещать его на сетке с визуальным предпросмотром, удалять ранее установленные здания и сохранять результат между игровыми сессиями.

## Описание проекта
Основной функционал:
- Размещение зданий по гри-сетке, предпросмотр здания перед установкой, запрет размещения поверх других зданий, подсветка области размещения;
- Удаление зданий, визуальное отображение активного режима удаления, обновление сохранённых данных;
- сохранение и загрузка размещенных зданий - данные сохраняются между сессиями в формате JSON.

Проект построен по принципам модульной архитектуры с использованием Assembly Definition (asmdef). Точка входа реализована через Bootstrapper, Bootstrapper создаётся из префаба и помечается как DontDestroyOnLoad.
В проекте используются New Input System, Unity Grid, Assembly Definition Files, JSON Serialization.
